# BlackJack
Blackjack console game written in DotNetCore with computer simulation

#Dependencies
[DotNet](https://www.microsoft.com/net/core) core V 1.0.1.

#Running
```
cd src/BlackJack
dotnet restore
dotnet run
``` 

#Tests
Test are written with NUnit using NSubstitute as the mocking library
```
cd test
dotnet restore
dotnet test
```
